# Subin on Ice

A personal figure-skating rink review archive.

## Publish with GitHub Pages

1. Upload every file in this folder to the root of the `subin-on-ice` repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose the `main` branch and the `/ (root)` folder, then click **Save**.
5. The site will appear at `https://SUBINonICE.github.io/subin-on-ice/` after GitHub finishes publishing it.

## Add another rink later

Add a new rink object inside `rink-data.js`. The home-page statistics, archive card, and map pin are created automatically.

## Add photos later

The first version uses designed photo placeholders. Photos can be added later by creating an `assets` folder and adding an image field to each rink in `rink-data.js`.
