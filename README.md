# Subin on Ice

A personal archive of ice rinks reviewed by an adult figure skater.

## Pages

- `index.html` — map, archive, rankings, and introduction
- `review.html` — dynamically generated individual rink review pages
- `about.html` — Subin's skating story and archive mission

## Adding a rink

Add a new rink object to `rink-data.js`. The homepage counts, place browser, map pins, review cards, and individual review page will update automatically.
